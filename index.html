Project Overview

Codeon is a smart contract auditing and analysis platform designed to help developers evaluate security, performance, and compliance before deployment. It provides automated multi-language support, runs a full suite of static and functional tests, and generates sleek PDF reports with detailed findings, charts, and actionable insights.

Core Objectives

Multi-language analysis

Solidity, Vyper, Rust, Move, Cairo, and more.

Comprehensive testing

Vulnerabilities, gas efficiency, compliance, and logic.

Professional reports

PDFs with charts, severity levels, and recommendations.

Simple frontend

Upload contracts, run audits, download results.

Scalable platform

Built for indie devs and enterprises.

Key Features

Security vulnerability checks

Reentrancy, overflow/underflow, access control flaws, unchecked calls.

Gas & performance optimization

Detect inefficient loops and high-gas operations.

Compliance & standards

ERC‑20, ERC‑721, ERC‑1155 conformance checks.

Logic & functionality

Detect DoS, front‑running, dead code, misuse of self‑destruct.

Deployment & upgradeability

Check delegatecall risks, proxy patterns, constructor safety.

PDF reports

Executive summaries, severity levels, charts, recommendations.
